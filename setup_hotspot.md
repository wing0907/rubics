# PC 유선 → 모바일 WiFi 핫스팟 공유

## Windows 설정에서 모바일 핫스팟 활성화

1. **설정 > 네트워크 및 인터넷 > 모바일 핫스팟** 이동
2. **"모바일 핫스팟" 켜기**
3. **"인터넷 공유" 설정:**
   - 공유 대상: `이더넷` (유선 네트워크)
   - 공유 방법: `WiFi`

4. **핫스팟 이름(SSID)과 비밀번호 확인**
   - SSID: `xxxx` (기억하기)
   - 비밀번호: `xxxx` (기억하기)

## 모바일 연결

1. 모바일 WiFi 설정에서 PC의 핫스팟 SSID 선택
2. 비밀번호 입력
3. 모바일에서 `http://localhost:8501` 또는 `http://192.168.137.1:8501` 시도

---

## 또는 옵션 B: USB를 통한 인터넷 공유

1. USB 케이블로 모바일 ↔ PC 연결
2. **설정 > 네트워크 및 인터넷 > USB 테더링** 활성화
3. 모바일이 PC의 인터넷 공유

---

## 또는 옵션 C: ngrok 사용 (권장)

만약 인터넷이 있다면 ngrok으로 인터넷 터널 생성:
```bash
python -m pip install pyngrok
python -c "from pyngrok import ngrok; ngrok.connect(8501)"
```

공개 URL이 생성되어 모바일에서 전 세계 어디서나 접속 가능합니다.
